## Requirements- E-Commerce Product Page

## Hardware Requirements

* Any personal computer or laptop

* Minimum 2 GB RAM

* Keyboard and mouse

* Optional internet access

## Software Requirements

* Windows / Linux / macOS

* Any modern web browser

* Code editor (VS Code / Sublime / Notepad++)

## Technologies Used

* HTML for page structure

* CSS for layout and styling

* JavaScript for quantity control and price calculation

## Functional Requirements

* Display product details

* Allow quantity and variant selection

* Update price dynamically

* Add product to cart and wishlist

* Show delivery estimate

* Toggle product specifications

## Non-Functional Requirements

* User-friendly interface

* Fast response to user actions

* Clean and readable code structure

* Browser compatibility

* Maintainability and scalability
